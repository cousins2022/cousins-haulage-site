COUSINS HAULAGE LTD WEBSITE - SIMPLE UPLOAD GUIDE

WHAT'S IN THIS FOLDER
- A complete website ready for Vercel
- Multi-page-style location pages:
  /manchester
  /london
  /birmingham
  etc.

WHAT YOU DO NEXT

1) CREATE A GITHUB ACCOUNT
   Go to https://github.com and sign up.

2) CREATE A NEW REPOSITORY
   Name it something like:
   cousins-haulage-site

3) UPLOAD THESE FILES
   Open your new repo and use Add file > Upload files.
   Upload everything from this folder.

4) CREATE A VERCEL ACCOUNT
   Go to https://vercel.com and sign in with the same GitHub account.

5) IMPORT THE REPO INTO VERCEL
   In Vercel:
   - Click Add New Project
   - Choose your GitHub repo
   - Click Deploy

6) CONNECT YOUR DOMAIN
   In Vercel:
   - Open your project
   - Go to Settings > Domains
   - Add cousinshaulageltd.co.uk

   Vercel will then show the DNS records you need.

7) ADD DNS RECORDS WHERE YOU BOUGHT YOUR DOMAIN
   Log in where you bought cousinshaulageltd.co.uk
   Open DNS / Name Server settings
   Copy the DNS records Vercel tells you to add

8) WAIT FOR DNS TO UPDATE
   This can take anywhere from a few minutes to up to 24 hours.

9) OPEN YOUR DOMAIN
   Your website should load on cousinshaulageltd.co.uk
